
# FlySafe — Flight Assurance Store

[![Deploy to GitHub Pages](https://github.com/xegea/flysafe/actions/workflows/deploy.yml/badge.svg)](https://github.com/xegea/flysafe/actions/workflows/deploy.yml)

This is a minimal Next.js + TypeScript scaffold for FlySafe — a small site to sell flight cancellation and lost luggage assurances.

Getting started

1. Install dependencies

```bash
npm install
```

2. Run development server

```bash
npm run dev
```

3. Open http://localhost:3000

What’s included

- Home page
- Products list (Cancellation, Lost Luggage)
- Product detail pages
- Mock checkout flow
- Basic responsive layout
